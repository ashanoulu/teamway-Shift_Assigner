# Teamway - Shift Assigner

### Java - 17.0
### Spring Boot - 3.0
### MySql - 8.0

## Steps to getting started

1. Change the database credentials in application.properties.
2. Run `migration.sh` to setup database
3. If you use maven to build use following
```bash
mvn spring-boot:run
```
